#Todo App (React + Node + PostgreSQL)

---

##Features

### Frontend (React)
- Add, update, delete todos
- View todo details in popup modal
- Search todos by title
- Filter by status (Completed / Incomplete)
- Sort by name or status
- Responsive grid layout (5 cards per row)
- React Hook Form validation
- Toast on add,delete,update

### Backend (Node + Express)
- REST APIs for todos
- PostgreSQL integration
- Soft delete (deleted flag instead of hard delete)
- Clean API structure

---

##Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- React Hook Form
- React Router
- Axios
- React Icons
- React-Toastify

### Backend
- Node.js
- Express.js
- PostgreSQL (pg module)
